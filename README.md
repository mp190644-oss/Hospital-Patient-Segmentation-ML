# AI-Based Hospital Patient Segmentation and Analytics

## Project Overview

This project applies Data Analysis, Data Visualization, and Machine Learning techniques to hospital patient data. The objective is to identify patient groups with similar characteristics and provide healthcare insights that can support hospital decision-making.

## Dataset

The dataset contains hospital patient records, including demographic information, clinical measurements, and healthcare-related attributes.

## Project Workflow

1. Data Collection and Loading
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Feature Engineering
6. Patient Segmentation using K-Means Clustering
7. Cluster Analysis and Interpretation
8. Healthcare Insight Generation

## Data Analysis Performed

* Missing value analysis
* Statistical summary of patient data
* Feature distribution analysis
* Correlation analysis
* Patient characteristic comparison

## Data Visualizations

* Histograms
* Bar Charts
* Box Plots
* Correlation Heatmaps
* Cluster Distribution Charts
* PCA-Based Cluster Visualization

## Machine Learning Techniques

### K-Means Clustering

Used to segment patients into multiple groups based on similarities in their healthcare characteristics.

### Patient Risk Categorization

Patients were grouped into categories such as:

* Low Risk
* Moderate Risk
* High Risk
* Critical Care
